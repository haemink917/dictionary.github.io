12323 김혜민
정보보호전문가 지망
이 사전은 정보보안 분야에서 사용되는 단어를 모아 놓았습니다.

정렬 기준
10개의 단어를 크게 네트워크, 보안, 암호의 3가지 분야로 나누었습니다. 각 분야 안에서 단어는 한국어 단어 먼저, 영어 단어는 뒤에 나오고 각각 ㄱㄴㄷ, abc 순서로 나열됩니다.

사전의 특징
2가지 정의로 단어 하나를 설명합니다. 첫번째는 제가 인터넷에서 찾은 단어의 뜻이고, 두번째는 제가 직접 작성한 뜻과 기본적인 원리입니다.
또한 몇 단어의 경우, 실생활 예시를 아래에 적어 더욱 이해하기 쉽도록 만들었습니다.
